instructions

Loading this project creates a Database schema called Football. The schema has 3 tables:

  • League which represents a league season.
  • Team which represents a football club.
  • Fixture which represents details of a football match.

The database is created empty. The status bar at the bottom of this page should show:
seasons: 0 teams: 0 fixtures: 0

The data we will be using is taken from another REST api.
So, first we need to fetch it from there to our database.
In order to load the data:

  1. click the link Load Data
  2. Enter the token sent to you in the mail
  3. Push one of the buttons and see how the status changes

The call to the web service is asynchronous.
So you would see the web service status changes to "success", when the call to the web service is done, before the results are back.
The status below that shows the number of records in each of the tables will change when the data is actually loaded.

After loading the data, you can test the next two links, showing the standings (league table), and team vs. team fixtures.
